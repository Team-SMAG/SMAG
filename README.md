## Workflow
주 1회 화상회의를 통한 Weekend 작업 현황 보고 <br>
개인 작업 및 디테일한 내용은 Private Repository 로 관리 <br>
학습한 내용은 잘 정리해서 공유할 예정 (TBD) <br><br>

__Client__ <br>
게임 플레이 담당 <br>
개발 : Unity, C# <br>

__Server__ <br>
API 제공 및 AI 연산부분 담당 <br>
개발 : AWS, Java, Spring <br>
AI : Rule-Based (no machine learning) <br><br>
